### Prerequisites

**Node version 18.x**

### Cloning the repository

```shell
git clone https://github.com/aditya-2010/vapordesk.git
```

### Install packages

```shell
npm install
```

### Setup .env file

```js
VITE_APP_AWS_ACCESS_KEY_ID=
VITE_APP_AWS_SECRET_ACCESS_KEY=
VITE_KEY_PAIR=
```

### Start the app

```shell
npm run dev
```
